# Developer Job Market Shift Analyzer

> A data analytics project analyzing 123K+ developer job postings to understand hiring trends, skills, and salaries.

---

## 🚧 Project Status

**Current Stage:** In Progress  

### ✅ Completed
- Data collection and setup (LinkedIn postings, skills, salaries, layoffs)
- Data cleaning pipeline
  - `01_clean_raw_data.py`
  - `02_extract_skills.py`
  - `03_clean_salary_layoffs.py`
- Processed datasets generated
- Exploratory Data Analysis (EDA)
  - Jupyter notebook with multiple visualizations
  - Initial trends identified (roles, skills, salary patterns)
